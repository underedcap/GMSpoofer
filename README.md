#**GMSpoofer**

Описание / Description: Лёгкий плагин для Minecraft, который позволяет быстро менять игровой режим с помощью команды /gm <0|1|3>. Я не видел ни одного нормального релиза без if/switch и решил сделать свой. A lightweight Minecraft plugin for Minecraft that allows quick game mode switching using /gm <0|1|3> without admin permissions. I hadn't seen a proper release without if/switch, so I decided to make one.

**#:Автор / Author: drugaddict:**

**:API-версия / API Version: 1.16/java16:**

**Команды / Commands:**

/gm 0 — Survival / Survival

/gm 1 — Creative / Creative

/gm 3 — Spectator / Spectator

**Установка / Installation:** Скопировать GMSpoofer.jar в папку plugins сервера и перезапустить. Copy GMSpoofer.jar to the server plugins folder and restart.

Примеры / Examples: /gm 0 — Survival, /gm 1 — Creative, /gm 3 — Spectator. /gm 0 — Survival, /gm 1 — Creative, /gm 3 — Spectator.
